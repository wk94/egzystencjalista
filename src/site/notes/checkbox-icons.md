# Minimal theme checkbox icons

**The following checkbox styles are available**

| Syntax | Description |
| ------ | ----------- |
| `- [ ]` | to-do |
| `- [/]` | incomplete |
| `- [x]` | done |
| `- [-]` | canceled |
| `- [>]` | forwarded |
| `- [<]` | scheduling |
| `- [?]` | question |
| `- [!]` | important |
| `- [*]` | star |
| `- ["]` | quote |
| `- [l]` | location |
| `- [b]` | bookmark |
| `- [i]` | information |
| `- [S]` | savings |
| `- [I]` | idea |
| `- [p]` | pros |
| `- [c]` | cons |
| `- [f]` | fire |
| `- [k]` | key |
| `- [w]` | win |
| `- [u]` | up |
| `- [d]` | down |
